# S11 - Mantenimiento Predictivo

Notebook: `S11 Mantenimiento Predictivo 4.0 - Clasificación de Fallas Industriales (1).ipynb`

Resumen: Notebook de diagnóstico y clasificación de fallas industriales (EDA, preprocesado, modelado y visualizaciones). Incluye recomendaciones de discretización, contexto nacional y bibliografía resumida.

Instrucciones rápidas:
- Ejecutar en un entorno con Python 3.8+ (recomendado) y las dependencias en `requirements.txt`.
- Para ejecutar el notebook y regenerar figuras:

```bash
pip install -r requirements.txt
jupyter nbconvert --to notebook --execute "S11 Mantenimiento Predictivo 4.0 - Clasificación de Fallas Industriales (1).ipynb" --output executed_notebook.ipynb --ExecutePreprocessor.timeout=600
```
